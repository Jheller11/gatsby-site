Variables:

```
x = 5
y = 'hello'
```

Conditional:

```
if true
    // do something
elsif true
    // do something
else
    // do something else
end
```

Loops:

```
x = 1
while x > 0
    puts x
    x -= 1
end
// 1
```

```
for x in 1...100
    puts x
end
```

```
array = []
array.each do |item|
    puts item
end
```
